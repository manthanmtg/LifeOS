# Habits

Track daily and weekly habits with streaks and completion history.

## Features

- Create habits with custom frequency (daily, weekly, specific days)
- Visual streak tracking to maintain momentum
- Check off habits each day with a simple toggle
- View completion history and consistency over time
- Track multiple habits simultaneously

## How to Use

Add the habits you want to build or maintain. Each day, open the module and check off the habits you completed. The streak counter shows your current consecutive completions — try not to break the chain. Review your weekly and monthly consistency to identify which habits are sticking and which need adjustment.

## Tips

- Start with 2-3 habits maximum; add more only once those are solid
- Don't aim for perfection — 80% consistency is still excellent
- If you break a streak, restart immediately; the gap matters less than the trend
