# Todo

A simple, no-frills todo list for quick tasks that don't need project-level tracking.

## Features

- Add tasks quickly with just a title
- Mark tasks as complete with a single click
- Optional due dates and priority levels
- Filter between active and completed tasks
- Clean, minimal interface designed for speed

## How to Use

Type a task and hit enter to add it. Check it off when done. That's it. Use this for quick, standalone tasks — groceries to buy, calls to make, errands to run. For larger, multi-step work, use the Compass module instead.

## Tips

- Keep this list short and actionable — if it grows past 15-20 items, move some to Compass
- Clear completed tasks periodically to keep the list focused
- Use due dates sparingly, only when a task has a real deadline
