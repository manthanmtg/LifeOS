# Rain Tracker

Log daily rainfall measurements and track precipitation patterns over time.

## Features

- Record daily rainfall in millimeters with date and optional notes
- Monthly and yearly rainfall totals with visual charts
- Compare rainfall across months and years
- Track cumulative rainfall for the current season
- Notes for weather observations (storm intensity, dry spells, etc.)

## How to Use

Each day it rains, log the measurement in millimeters along with the date. The module aggregates your entries into monthly and yearly summaries with charts. Use the comparison view to see how the current season stacks up against previous years.

## Tips

- Measure at the same time each day for consistency (morning is typical)
- Log zero-rainfall days during monsoon season if you want accurate rainy-day counts
- Pair this data with Crop History to correlate rainfall patterns with crop performance
