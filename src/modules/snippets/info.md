# Snippets

Store and organize reusable code snippets, commands, and technical notes.

## Features

- Save code snippets with syntax highlighting by language
- Tag snippets for easy categorization (bash, python, regex, SQL, etc.)
- Quick search across all snippets by title, tag, or content
- Copy to clipboard with one click
- Add descriptions explaining what each snippet does and when to use it

## How to Use

When you write or find a useful piece of code, command, or config, save it here with a descriptive title and the relevant language tag. Add a short description so future-you remembers the context. When you need it again, search by keyword or browse by tag.

## Tips

- Include a brief explanation of _why_ the snippet is useful, not just the code
- Save common terminal commands you always forget (SSH tunnels, git tricks, etc.)
- Use consistent tags so related snippets are easy to find together
