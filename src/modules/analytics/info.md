# Analytics

Track visitor analytics for your public-facing LifeOS pages.

## Features

- Page view tracking across all public routes
- Visitor counts with unique vs. total views
- Referrer tracking to see where traffic comes from
- Time-based charts showing traffic trends
- Lightweight, privacy-respecting — no third-party scripts

## How to Use

Analytics runs automatically on your public pages. Visit this module to view the dashboard showing page views, unique visitors, and traffic sources. Use the date range filters to analyze trends over different time periods. No setup required beyond enabling the module.

## Tips

- Check referrer data to understand which channels drive the most traffic
- Compare traffic before and after publishing new blog posts to measure impact
- Use the data to decide which public modules are worth expanding
