# AI Usage

Track your AI API usage, costs, and token consumption across different providers and models.

## Features

- Log API calls with provider, model, token count, and cost
- Track spending across providers (OpenAI, Anthropic, Google, etc.)
- View usage trends over time with daily/monthly breakdowns
- Monitor token consumption by model
- Set budget awareness with running cost totals

## How to Use

Add entries for your AI API usage — either manually or by logging key sessions. Track which models you're using, how many tokens each call consumes, and the associated cost. Review the dashboard to understand your spending patterns and optimize which models you use for different tasks.

## Tips

- Log usage consistently to get an accurate picture of your AI spending
- Compare cost-per-task across models to find the best value for each use case
- Review monthly totals to stay within your budget
