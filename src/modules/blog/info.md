# Blog

Write and publish blog posts with a built-in markdown editor.

## Features

- Full markdown editor with live preview
- Publish or save as draft
- Tags and categories for organization
- Public-facing blog page with clean typography
- SEO-friendly URLs based on post titles

## How to Use

Create a new post, write your content in markdown, and add relevant tags. Save as a draft while working on it, then publish when ready. Published posts appear on your public blog page. You can edit or unpublish posts at any time from the admin view.

## Tips

- Use drafts liberally — start posts when inspiration hits and finish them later
- Add 2-3 relevant tags per post to help readers find related content
- Write a compelling opening paragraph; most readers decide to stay or leave within seconds
