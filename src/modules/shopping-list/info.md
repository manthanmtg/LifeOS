# Shopping List

A dedicated shopping list manager for groceries, household items, and other purchases.

## Features

- Add items quickly with name and optional quantity
- Categorize items (groceries, household, personal care, etc.)
- Check off items as you shop
- Reuse previous lists or items for recurring purchases
- Simple, mobile-friendly interface for use in-store

## How to Use

Add items as you run out of things or think of what you need. When you're at the store, open the list and check items off as you pick them up. Checked items move to the bottom so you can focus on what's left. Clear the list after shopping or keep checked items for easy re-adding next time.

## Tips

- Add items throughout the week as you notice you're running low
- Group items by store section (produce, dairy, etc.) to shop more efficiently
- Keep a "staples" set of items you buy every trip for quick list building
