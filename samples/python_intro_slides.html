What is Python?

  • High-level programming language
  • Created by Guido van Rossum (1991)
  • Simple and readable syntax
  • Used in web, AI, automation, data science

Why Learn Python?

  • Easy for beginners
  • Huge community support
  • Large number of libraries
  • Used by companies like Google, Netflix, Instagram

Your First Python Program


print("Hello, World!")

This prints text to the screen.

Python Variables


name = "Alice"
age = 25
print(name)
print(age)

Variables store data.

Python Data Types

  • int → 10
  • float → 3.14
  • string → "hello"
  • boolean → True / False
  • list → [1,2,3]

Python Example


for i in range(5):
    print(i)

This prints numbers from 0 to 4.
