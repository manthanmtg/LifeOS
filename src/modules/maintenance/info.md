# Maintenance

Track recurring maintenance tasks for your home, appliances, and other assets.

## Features

- Create maintenance tasks with frequency (monthly, quarterly, yearly, etc.)
- Track last completed date and next due date
- Categorize by area (home, kitchen, electrical, plumbing, garden, etc.)
- Log completion with optional notes and cost
- Overdue task highlighting so nothing gets missed

## How to Use

Add maintenance tasks that need periodic attention — AC filter cleaning, water tank servicing, pest control, appliance checks, etc. Set the frequency and last done date. The module calculates when each task is next due and highlights overdue items. Mark tasks complete as you do them to keep the schedule current.

## Tips

- Start with the most critical tasks (safety-related, expensive-if-neglected) and add more over time
- Review the overdue list weekly and batch similar tasks together
- Log costs to track your annual maintenance spending
